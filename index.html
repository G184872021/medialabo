「Webコンテンツ作成」の成果

本テーマの学習目的

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的です。到達目標は次の4つです。(1) 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる、（2）コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できる、（3）モーション解析・編集ができる、（4）HTML等を用いたWebページ作成ができる。なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

プロジェクト課題について（200字以上）

このプロジェクト課題は、WWWのサーバへ検索し、検索結果を表示するページを作成しました。私は、世界の天気のデータを表示するページを作成しました。このページでは、全12都市のどれか一つを選びその都市の緯度、経度、天気、最低気温、最高気温、湿度、風速、風向、都市名が表示されます。

この課題の工夫点として、第2回で学んだCSSを用いて背景色を変えました。また、第5回で学んだ、プルダウンメニューを使用し、ユーザがメニューに表示される都市名を一つ選択し検索結果を表示するようにしました。

プロジェクト課題で作成したページ:project.html

数当てゲーム（300字以上）

数当てゲームでは1～10の乱数を一つ決め、ユーザが答えを予想するゲームである。この課題では、最大3回予想を行います。回答回数が3回以内で予想が当たったら「正解です．おめでとう!」と表示します。回答回数が2回以下でかつ予想より答えが大きかった場合は「まちがい．答えはもっと大きいですよ」と表示させます。また、回答回数が2回以下でかつ予想より答えが小さかった場合は「まちがい．答えはもっと小さいですよ」と表示します。回答回数が3回目で外れだった場合「まちがい．残念でした答えは ◯ です．」と表示します。 もしユーザの回答回数が4回以上になった場合は「答えは ◯ でした．すでにゲームは終わっています」と表示させます。

数当てゲームのページ:kazuate.html

毎回の実験で作成したページ（100字以上）

第1回WWWとHTML

今回はHTMLについて学びました。HTMLはWeb ページの構造を記述するための書式で、HTML 文書を作ることによって、Web ページに掲載するコンテンツを構造化し、テキスト、画像、リンク、テーブル、フォームなどの要素を含めることができます。

演習の成果

  • 演習1-3 HTML文書の作成
  • 演習1-4 HTMLのさまざまな要素

第2回CSSによるWebページの装飾

今回は前回学んだHTMLの要素それぞれの見た目の設定（装飾）について学びました。CSSはHTMLと組み合わせることでWebページの文字の色やフォント、背景色、ボーダー、余白などのデザインをより美しく整えることができます。

演習の成果

  • 演習2-3 ウェブブラウザ説明ページの装飾

第3回JavaScriptの基本

今回はJavaScriptでできることについて学びました。JavaScript は Web ページ上で複雑な機能を実現するプログラミング言語です。 Web ページを開いた直後や、ボタンを押したときに JavaScript プログラムが実行されます。ページの自動更新やアニメーションなど動きのあるページを実現するために JavaScript プログラムを使います。

演習の成果

  • 演習3-6 オブジェクトの参照

第4回DOMとJavaScript

今回はDOMについて学びました。DOMはWeb コンテンツを表現するデータ構造です。DOM を使用することによって、JavaScript などのプログラムによって、Web コンテンツの内容を読み取ったり、コンテンツに変更を加えたりする処理が書けるようになります。

演習の成果

  • 演習4-2 ~ 4-5

第5回イベント処理

今回はイベント処理について学びました。イベント は Web ページ上で起こった動作や出来事のことです。例えば、ユーザーがある要素の上をマウスでクリックしたり、ある要素の上にカーソルを持ってくるなどのイベントが発生したとき、JavaScript プログラムが動作するような「イベント処理」の実現方法を学びました。

演習の成果

  • 演習5-2 イベントハンドラの登録を確認する
  • 演習5-3 イベントを選択する

第6回プロジェクト課題を完成させよう

今回はAJAX通信について学びました。AJAX はページ全体を再読み込みすることなく、ページの一部を更新する処理です。この処理では、JavaScript の XMLHttpRequest というオブジェクトを使った通信と、これまでに学んだ HTML、 CSS、 JavaScript、 DOM、 イベント処理の技術を組み合わせます。

苦心点、工夫点、強調点、感想（1000字以上）

第１回の授業ではHTMLのさまざまな要素について学びました。課題1-4では、習った要素を使ってHTMLの文書を作成しました。苦心点として、「演習1-3のHTMLファイル」と「index.html」をハイパーリンクにして表示させるところで少し苦労しました。また、画像の埋め込み部分が少し難しかったです。第２回の授業では、CSSによるWebページの装飾について学びました。演習2-3のウェブブラウザの説明ページでは、文字の色やフォント、背景の色、見出し部分の文字に少し陰を入れる、などをして見やすくなるようにページ作成をしました。工夫点として、４つのウェブブラウザの説明部分をグリッドレイアウトに変更するところで、グリッド同士の幅の部分の調整と高さを合わせるところを工夫しました。見出し部分の陰を入れるところでは、陰の幅を調整し、少しだけ陰を入れて強調させました。また、それぞれのウェブブラウザ名のところは、フォントを変え文字を太くし強調させました。第３回の授業ではJavaScriptの基本について学びました。演習3-6のオブジェクトの参照では、キャンパスの住所、D館の教室名、日本語の学科名の３つをコンソールに出力するプログラムを書きました。for-of構文をしようするところが難しかったです。第４回ではDOMとJavaScriptについて学びました。演習の4-4から4-5の課題ではプログラムの作成が全くわからなくてとても苦戦しました。第５回では、イベントの処理について学びました。表示されているボタンをクリックしたら背景の色が変わるなどいろんなイベント処理を学びました。第６回では、AJAX通信について学びました。AJAX通信はブラウザからサーバへデータを要求することとサーバからブラウザへデータを返す処理をします。Project課題では世界の天気が表示されるページを作りました。工夫点は、まず第２回で学んだCSSを使用し背景の色を変えました。また、プルダウンメニューを使用し、ユーザがメニュー欄に表示される都市を一つ選択し、検索ボタンを押すと選択した都市の経度や緯度、天気など９個のデータが表示されるようにしました。index.htmlの課題では、見出し、小見出し、小見出しの中の６個の見出し、背景色の色をそれぞれ変え、強調させ、見やすくなるようにしました。また、見出しごとに文字のサイズを変え強調させました。この授業をとおして、Webコンテンツを作成していく上で必要なHTML文書の作成などいろいろなことを学びました。プログラムを書き加えたり、書き換えたりなど難しいところがたくさんありました。
